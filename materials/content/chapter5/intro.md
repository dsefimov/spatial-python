# **Справочный материал**

В процессе изучения основ Python вы познакомились с различными типами данных, которые обладают достаточно объемным функционалом. Подробно рассмотреть все возможности каждого типа в рамках вводных материалов невозможно, да и заучивать их наизусть не требуется. Однако полезно знать о существовании этих методов, чтобы понимать, чем можно воспользоваться при решении конкретных задач.

В этом разделе собраны перечни методов, доступных для таких типов данных как: строки, списки, кортежи, диапазоны, словари, множества. Более полное описание всегда доступно в официальной документации (умение искать нужную информацию по документации — важный навык). Данный справочник в первую очередь предназначен для быстрого доступа к примерам использования различных методов.

* Строки: [перечень методов](../chapter5/22.str.ipynb) \ [документация](https://docs.python.org/3/library/stdtypes.html#text-sequence-type-str)
* Последовательности (списки, кортежи, диапазоны): [перечень методов](../chapter5/23.seq.ipynb) \ [документация](https://docs.python.org/3/library/stdtypes.html#lists)
* Словари: [перечень методов](../chapter5/24.dict.ipynb) \ [документация](https://docs.python.org/3/library/stdtypes.html#mapping-types-dict)
* Множества: [перечень методов](../chapter5/25.set.ipynb) \ [документация](https://docs.python.org/3/library/stdtypes.html#set-types-set-frozenset)
